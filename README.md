# ⚽ Mundial 2026 — Cuenta Atrás

Una landing page moderna, premium y totalmente responsive para seguir la cuenta atrás hacia el inicio del **Mundial de Fútbol FIFA 2026** (Estados Unidos, Canadá y México).

## ✨ Características

- **⏳ Contador Premium**: Cuenta atrás en tiempo real con animaciones fluidas.
- **🌓 Modo Oscuro / Claro**: Diseño adaptado para ambos modos con transiciones suaves.
- **📱 Totalmente Responsive**: Compatible con móviles, tablets y escritorio (incluye menú hamburguesa animado).
- **🇪🇸 Partidos de España**: Información detallada de los primeros partidos de la selección española (rivales, sedes, horarios y canales de TV).
- **📅 Integración con Calendario**: Botones directos para añadir los partidos de España a Google Calendar.
- **📡 Dónde Ver**: Sección dedicada con enlaces directos a las plataformas de retransmisión oficial en España (RTVE Play, DAZN, GolStadium, Movistar+).
- **✨ Experiencia de Usuario**: Pantalla de carga inicial (preloader) y efectos de partículas interactivas en el fondo.

## 🚀 Tecnologías Utilizadas

- **HTML5**: Estructura semántica y SEO optimizado.
- **CSS3 (Vanilla)**: Diseño moderno con variables CSS, Flexbox, Grid y animaciones personalizadas.
- **JavaScript (Vanilla)**: Lógica del contador, gestión de temas, animaciones de partículas y manejo del DOM.

## 🛠️ Instalación y Uso

No requiere dependencias externas ni compilación. Solo necesitas un navegador web.

1. Clona este repositorio o descarga los archivos.
2. Abre el archivo `index.html` en tu navegador favorito.
3. ¡Disfruta de la cuenta atrás!
